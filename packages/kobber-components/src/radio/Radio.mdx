# Radio Group and Inputs

A Radio Group contains more than one Radio Input, and the Radio Inputs are mutually exclusive. Radio Input must be used inside a Radio Group.

Use Radio Inputs to select only one option from a list of choices.

Keyboard navigation: Use tab to reach the radio group, and <a href="https://www.w3.org/WAI/ARIA/apg/practices/keyboard-interface/#:~:text=A%20primary%20keyboard,multiple%20focusable%20elements.">arrow keys</a> to move focus inside the group.
