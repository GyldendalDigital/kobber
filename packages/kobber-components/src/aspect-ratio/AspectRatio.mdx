# AspectRatio

Uses CSS `padding-top` instead of `aspect-ratio` to support [old browsers](https://caniuse.com/mdn-css_properties_aspect-ratio).

For example, `aspect-ratio: 16/9` will be converted into `padding-top: calc((9 / 16) * 100%)`.

`aspect-ratio` is [baseline since 2021](https://developer.mozilla.org/en-US/docs/Web/CSS/aspect-ratio), so this should be deprecated at some point.
